The authoritative list of variables is **`.env.example`** in the repository root.

| Variable                                              | Notes                                                                                                                                                                                            |
| ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |

| `SENTRY_DSN` / `SENTRY_*`                             | Optional frontend and backend Sentry ([Sentry and logger](https://github.com/DiegoFleitas/letterboxd-movie-justwatch/wiki/Sentry-and-logger)).                                                   |

| `JACKETT_API_KEY` / `JACKETT_API_ENDPOINT`            | Optional alternative search.                                                                                                                                                                     |
| `CACHE_TTL`                                           | Optional cache TTL override (seconds).                                                                                                                                                           |
| `PORT`                                                | Backend port (default **3000**).                                                                                                                                                                 |
| `E2E_API_BASE_URL`                                    | Playwright backend smoke base URL (default `http://127.0.0.1:3000`).                                                                                                                             |
