# Repository layout

| Path                         | Role                                                                                                                                                                                                                                                                                                                |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/client/`                | Vite app root: React (`src/`), static assets, styles                                                                                                                                                                                                                                                                |
| `src/server/main.ts`         | Process entrypoint (Fastify + Sentry bootstrap)                                                                                                                                                                                                                                                                     |
| `src/server/createServer.ts` | Fastify wiring: routes, static files, sessions                                                                                                                                                                                                                                                                      |
| `src/server/controllers/`    | HTTP handlers (lists, search, posters, proxy, Jackett)                                                                                                                                                                                                                                                              |
| `src/server/lib/`            | Shared backend utilities (axios, Redis, Letterboxd scraping, canonical providers, schemas, PostHog); domain/API types in `src/server/lib/types/`                                                                                                                                                                    |
| `redis/`                     | Redis image/entrypoint for Docker Compose; `data/` (JSON cache seeds); `scripts/` (export/seed/validate snapshot, build canonical providers). → [Redis and local dev](Redis-and-local-dev)                                                                                                                          |
| `scripts/`                   | Letterboxd fixture updates and other one-off utilities                                                                                                                                                                                                                                                              |
| `tests/`                     | Vitest unit and integration tests. → [Tests](Tests), [Test fixtures](Test-fixtures), [Test goldens](Test-goldens)                                                                                                                                                                                                   |
| `tests/e2e/`                 | Playwright (UI mocks + backend smoke). → [E2E Playwright](E2E-Playwright)                                                                                                                                                                                                                                           |
| `docs/`                      | Markdown under [`docs/wiki/`](https://github.com/DiegoFleitas/letterboxd-movie-justwatch/tree/master/docs/wiki) is the in-repo copy of this wiki; **[Wiki publish](https://github.com/DiegoFleitas/letterboxd-movie-justwatch/blob/master/docs/Wiki-publish.md)** describes the git sync. Wiki entry: [Home](Home). |

**Import aliases** (see root `tsconfig.json`, `src/client/tsconfig.json`, `vite.config.ts`, `vitest.config.ts`): `@server/…` → `src/server/…`; `@/…` → `src/client/src/…` (client app modules only).
