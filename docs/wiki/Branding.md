# Branding

## README banner

The image at the top of the main **README** is stored in the repository at:

`docs/images/github-banner.png`

The README renders it with HTML (`<img width="800">`) so it stays centered, scales on narrow viewports, and uses a light corner radius and shadow. Update the PNG in a pull request when refreshing the artwork.

## GitHub social preview

For a consistent card when the repository link is shared:

1. Open the repository on GitHub.
2. Go to **Settings → General → Social preview**.
3. Upload the same image as the README banner (or a **1280×640** crop).

GitHub’s [documentation on social images](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/customizing-your-repositorys-social-media-preview) describes recommended dimensions.
